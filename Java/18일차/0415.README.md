Collection Framework
	- 자바가 자료구조를 구현해 놓은 것 
	- 자바의 모든 자료구조 구현 클래스는 Collection인터페이스를 구현한 클래스이다.
	- 자바 자료구조 특징
		- 객체만 저장할 수 있다.
		- 크기가 가변적이다.
		- 다양한 메소드를 지원한다.
