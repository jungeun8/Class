id와 class 속성

id속성은 특정 태그를 식별하기 위해서 사용된다.

html 문서 전체에서 id속성 값은 고유한 값이어야 한다.

id속성값은 특수문자나 숫자로 시작할 수 없다.

class속성은 같은 클래스 속성값을 가진 태그에 같은 스타일을 적용하기 위해 사용된다.

html 문서에 같은 class속성값을 가지는 태그는 여러 개 존재할 수 있다.

class 속성은 한 번에 여러개 class 속성값을 가질 수 있다.
