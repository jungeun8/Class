# HTML(21.05.12)
Html 
- 부모태그안에 자식태그를 적어야한다. 
 - 태그는 속성과 함꼐 적어야한다. 
- 속성은 태그와 관련된 부가적인 정보, 추가기능을 정의한다.
- 형식 : <태크명 속성명 = "속성값" 속성명 = " 속성값"> 
- 속성명과 속성값은 붙여서 적고, 속성값은 "" 사이에 적는다.
- 동일한 속성명을 여러 번 적을 수 없다.
- 속성명을 적을 때는 특별한 순서가 없다. 

-속성
- 태그마다 고유한 속성을 가지고 있다.

- head 태그 안에 웹브라우저가 html문서를 해석할때 필요한 정보를 정의하는 곳
- body 태크 안에 실제로 화면에 표시되는 내용이 정의되는곳 

태그
- 컨텐츠를 담는 그릇이다.
- 컨텐츠의 종류에 따라서 적절한 태그를 사용해야한다.
